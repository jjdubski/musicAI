# How to run 

## Windows:  

### python -m venv venv
### .\venv\scripts\activate
### pip install -r requirments.txt
### python main.py
##### Let it run and complete. Then do.
### python convert.py

## MacOS or Linux:  

### python3 -m venv venv
### source ./venv/bin/activate
### pip install -r requirments.txt
### python main.py
##### Let it run and complete. Then do.
### python convert.py
